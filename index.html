Mapping Projects from New Maps Plus Program

UKY North Farm

Created a reference map to display field bourdries for Plant & Soil Science Department at the University of Kentucky. Current PI's name is display with popup and field number displayed in boundaries for easy identification. Map made using Carto.

North Farm

KY Hemp

Historic production of hemp produced in Kentucky using USDA Ag Census data from 1850, 1900, and 1910. Please note that scales are different units for each year and is almost half as much for each Census. The percentage of change map is not complete.

About the Maker

Thanks for visiting my Map Portfolio page. Here is a little bit about me. My name is Gene Hahn and enjoy making maps for others. I currently work in the Precision Ag field creating Variable Rate Application maps for Nitrogen application in corn. I have experienced the hardships of learning several different mapping software packages like ArcMap, QGIS, and FarmWorks. So to further develop skills in mapping I join the New Mapping Plus program at the University of Kentucky. For more information please visit GitHub profile.
